# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺让所有参与者都能获得尊重和包容的体验，无论年龄、体型、肤色、能力、学历、性别认同、经验水平、国籍、个人外貌、种族、宗教或性认同和取向。

## 我们的标准

### 鼓励的行为

- 使用欢迎和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 与社区其他成员表现出同理心

### 不可接受的行为

- 使用带有性别歧视、种族主义、仇外心理或歧视性语言
- 人身攻击、贬低他人或进行人身攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如物理地址或电子地址）
- 其他可以被合理地认为是不专业或不适当的行为

## 责任范围

社区管理者负责明确可接受行为的标准，并应采取适当和公平的纠正措施来应对任何不可接受的行为。

社区管理者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献，并在适当时通报原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间代表社区的情况。

代表社区包括使用官方电子邮件地址、通过官方社交媒体账号发布，或作为指定代表在在线或离线活动中行事。

## 执行

可以通过以下方式联系社区管理团队：issues@xiaoye-components.github.com

所有投诉都将得到及时和公平的审查和调查，并给予保密处理。

社区管理团队有义务对任何事件的相关方保密。进一步执行细节的细节可能会单独提供。

## 处罚措施

如果社区成员有不不当行为，社区管理者可以采取以下任何措施，形式由社区管理者自行决定：

- 口头警告
- 书面警告
- 临时禁止参与社区活动
- 永久禁止参与社区活动
