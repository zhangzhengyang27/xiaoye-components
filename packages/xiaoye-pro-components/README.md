# xiaoye-pro-components

`xiaoye-pro-components` 是基于 `xiaoye-components` 的中后台业务增强组件库。

## 安装

```bash
npm install xiaoye-pro-components vue
```

## 使用

```ts
import { createApp } from "vue";
import XiaoyeProComponents from "xiaoye-pro-components";
import "xiaoye-pro-components/style.css";

const app = createApp({});
app.use(XiaoyeProComponents);
```

仓库与文档：<https://github.com/zhangzhengyang27/xiaoye-components>
