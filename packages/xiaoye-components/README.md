# xiaoye-components

`xiaoye-components` 是一个面向中后台系统的 Vue 3 组件库。

## 安装

```bash
npm install xiaoye-components vue
```

## 使用

```ts
import { createApp } from "vue";
import XiaoyeComponents from "xiaoye-components";
import "xiaoye-components/style.css";

const app = createApp({});
app.use(XiaoyeComponents);
```

仓库与文档：<https://github.com/zhangzhengyang27/xiaoye-components>
