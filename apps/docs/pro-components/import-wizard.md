# ImportWizard 导入向导

`xy-import-wizard` 适合承接“上传文件 -> 校验 -> 查看结果”这类导入流程，把阶段切换做成稳定向导。

## 基础用法

:::demo 当前示例先演示步骤流和阶段内容承接，后续可以再接上传和结果表格。
pro/import-wizard/basic
:::

## 当前定位

- 面向导入流程的向导型容器。
- 统一步骤条、阶段内容和上一步下一步动作。

## 当前边界

- 当前不包含文件上传、解析进度和失败修复工作流。
- 真实导入逻辑仍需页面层或专门上传组件承接。
