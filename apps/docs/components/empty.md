# Empty 空状态

`xy-empty` 用来承接列表为空、筛选无结果和首次进入时暂无内容的页面状态。它关注的是“信息兜底 + 下一步动作”这条链路。

## 基础用法

:::demo 最基础的空状态只需要标题和描述，就能让列表页不至于直接落到空白区域。
empty/basic
:::

## 带操作入口

:::demo 当页面为空时，通常最好给出一个明确的下一步动作，例如创建、查看模板或重置筛选。
empty/action
:::

## 何时使用

- 列表页当前没有任何数据。
- 筛选条件过严，结果集为空。
- 首次进入某个模块，需要引导用户完成第一步操作。

## API

### Empty Attributes

| 属性          | 说明     | 类型     | 默认值                     |
| ------------- | -------- | -------- | -------------------------- |
| `title`       | 标题文案 | `string` | `'暂无数据'`               |
| `description` | 描述文案 | `string` | `'这里还没有可展示的内容'` |

### Empty Slots

| 插槽      | 说明                 |
| --------- | -------------------- |
| `default` | 底部操作区或补充说明 |
